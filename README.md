Kelompok : UAS Mobile Programing membuat apk headphone dengan menembak sebuah API dari Modul yang diberikan oleh pak Puspa

Untuk jalankan program pastikan sudah menginstall Android Studio & IntelliJ IDEA

Lalu, untuk menjalankan program, jalankan perintah berikut:
cukup run setelah git clone repo dan masuk ke dalam Android Studio & IntelliJ IDEA

Terdapat beberapa fitur di UAS-API, yaitu;
1. Menembak sebuah server dengan API
2. Melihat seluruh isi event
3. Dapat Menghapus & Mengedit event
4. Simpan

Penggunaan APK API event:
1. Langsung masuk semua event yang akan mendatang, selesai, dll
2. Mengedit sebuah event dengan sebuah jadwal, jam, daerah, dll
3. Menghapus sebuah event
4. Menyimpan editan

1. User harus mengetahui letak sdk disimpan
2. User harus menyalin letak directory
3. User harus membuat txt kosong dengan nama local.properties atau bisa mencopy gradle yang memiliki .properties
4. User harus mengcopy kalimat diatas dan hanya mengganti sdk.dir=C\:\\Users\\ NAMA FILE LAPTOP \\AppData\\Local\\Android\\Sdk dengan file atau directory sdk disimpan
5. User bisa mengedit di notepad agar lebih mudah dan bisa langsung disimpan
